# smelly-tofo
### 臭豆腐外卖平台
#### 系统板块划分
分为三大板块：
* 用户模块：用于用户点餐、修改个人信息、订单追踪
* 商家模块：用于菜单管理、营业统计、订单管理与追踪
* 配送板块：配送员获取订单信息，调整订单配送状态，修改配送员信息与绑定健康枕使用。
* 后台管理：管理各类商家的入住与工商信息等，管理用户信息与用户等级，同时具有数据看板功能，能够统计每日平台注册人数、日活人数、各类营业额等数据。
